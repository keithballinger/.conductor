# .conductor
Markdown files for managing a coding AI Agent.

Example in examples folder.

`conductor.md` is a document you can feed into the Gemini CLI to help you create these files. 

Example getting started:
1. Create a new folder and cd into it.
2. launch gemini cli `$ gemini`
3. Enter a prompts like: "I'd like to build a website for uploading pictures of dogs. Use the process outlined in `conductor.md` to set everything up.
4. Review the files it creates and adjust as needed.
5. Relaunch the gemini cli with the prompt in `prompt.md`
